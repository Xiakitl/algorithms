# Current implemented algorithms

## Sorting
* Insertion sort O(n²)
* Selection sort O(n²)

## Maximum Subarray
* Divide and conquer O(n log n)
* Kadane's algorithm O(n)

## Square Matrix Multiplication
* Simple algorithm O(n³)
* Divide and conquer O(n³)
